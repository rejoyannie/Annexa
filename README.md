# Simple Static Portfolio

This is a minimal responsive static site (HTML/CSS/JS) ready for GitHub Pages.

How to use:
1. Update index.html with your name, projects, and contact details.
2. (Optional) Replace Formspree action with your own form endpoint, or remove the form.
3. Deploy to GitHub Pages (instructions below).

Deployment (quick)
- Create a new repository on GitHub (e.g., `yourusername/portfolio`).
- Locally:
  git init
  git add .
  git commit -m "Initial site"
  git branch -M main
  git remote add origin https://github.com/yourusername/portfolio.git
  git push -u origin main

- On GitHub: Settings -> Pages -> Source: main branch / root -> Save.
- Site URL: https://yourusername.github.io/portfolio (or https://yourusername.github.io if repo is username.github.io).
